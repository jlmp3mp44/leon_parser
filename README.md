Парсер даних LeonBets 
Інструмент для автоматизованого збору та структурування даних з букмекерської платформиLeonBets.

 Основні характеристики
Парсинг : Витягує повну ієрахію даний: від спорту до характеристик окремого маркету

Багатопотоковість : Використання пулу з 3 потоків для паралельного розрахунку матчів та маркетів, що значно пришвидшує роботу.

Чиста архітектура : Розподіл на DTO (об'єкти для десеріалізації JSON) та Моделі (моделі для бізнес-логіки).

Формат UTC : Усі позначки часу автоматично конвертуються у стандарт UTC.

Моделі — очищені та корисні структури, з якими працює базова логіка застосунку.

 Формат виводу в консоль
Дані відображаються у зручному деревоподібному форматі:

### Приклад виводу в консоль:
```text
Спорт, Ліга
└──  Назва матчу |  Дата+Час UTC | ID
    ├──  Назва ринку (Market)
    │   ├──  Назва ісходу | Коефіцієнт | ID
    │   └──  Назва ісходу | Коефіцієнт | ID
    └──  Назва наступного ринку
        └──  Назва ісходу | Коефіцієнт | ID
```
 Швидкий старт
Щоб запустити проект локально, виконайте такі дії:

1. Клонування репозиторію
git clone https://github.com/jlmp3mp44/leon_parser.git
cd leonbets-parser
2. Запуск
Відкрийте проєкт у вашому IDE та запустіть файл: src/Main.java
Після запуску ви побачите структурований потік даних на консолі в режимі реального часу.

Паралелізм: Пул потоків (3 потоки)

Розбір даних: JSON (відображення DTO)

Часовий пояс: UTC Standard
